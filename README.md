# Node Plantilla Base

Proyecto inicial con estructura modular para una API REST.

## Estructura del Proyecto
- **src/index.js**: Punto de entrada de la aplicación.
- **src/routes/**: Definición de los endpoints.
- **src/controllers/**: Lógica de las funciones.
- **src/models/**: Modelos de datos.
- **src/middlewares/**: Funciones de control intermedias.
- **src/config/**: Configuraciones generales.

## Comandos
- `npm run dev`: Inicia el servidor en modo desarrollo con Nodemon.
- `npm start`: Inicia el servidor en modo producción.
